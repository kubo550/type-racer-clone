# Type Racer Game 

My version of popular online game [typeracer](https://play.typeracer.com/) build in **Next.js** with firebase in TypeScript.

- React Context 
- REST API
- Firebase Auth 
- Yup Validator 
- Framer Motion 
- NProgress
- Multilanguage
